# Simulation Purpose

This project exists to support theoretical learning about group combat behaviour.

The goal is to encode the factors that make a group effective as a fighting force, then run deterministic scenarios against different enemy profiles to observe likely strengths, weaknesses, and failure points.

The simulation is not expected to perfectly predict real Empire battles.

The simulation should prioritise:

- recognisable individual behaviour
- group cohesion and morale effects
- role synergy
- pressure, hesitation, discipline, and confidence
- inspectable outcomes
- deterministic replay
- useful after-action analysis

The simulation should avoid:

- video-game HP sponge logic
- random block chance as the primary defence model
- omniscient tactical AI
- visual spectacle that cannot explain itself
- mechanics that cannot be tested headlessly
